# SV detection pipeline

Directory structure, to be adjusted as necessary

* `data/`  
    Data - only config files are version controlled
    * `config/`  
        Config files 
    * `raw_vcfs/`  
        Raw VCFs
    * `std_vcfs/`  
        Standardized VCFs
* `logs/`  
    LSF logs
* `scratch/`  
    Intermediate files
* `scripts/`  
    Scripts invoked by workflows
* `workflows/`  
    Snakefiles
